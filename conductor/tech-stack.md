# Technology Stack

## Core Frameworks
*   **Language:** PHP 8.2+
*   **Backend Framework:** Laravel 12.0
*   **Frontend/Admin Panel:** Filament v4
*   **Real-time UI:** Livewire 3
*   **Interactivity:** Alpine.js
*   **Styling:** Tailwind CSS

## Primary Packages
*   **Authorization:** `spatie/laravel-permission` (Role-Based Access Control)
*   **Excel/CSV Management:** `maatwebsite/excel` (Import/Export functionality)
*   **Development & Debugging:** `laravel/tinker`, `laravel/boost`

## Testing Stack
*   **Testing Framework:** Pest 4
*   **Laravel Support:** `pestphp/pest-plugin-laravel`

## Infrastructure (Inferred)
*   **Database:** MySQL 5.7+ / MariaDB 10.2+
*   **Package Managers:** Composer (PHP), NPM (JavaScript)
*   **Build Tool:** Vite
