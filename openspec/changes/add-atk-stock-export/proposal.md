# Proposal: Export ATK Stock Request and Bulk Export ATK Stock Request

This proposal outlines the implementation of two new features:
1.  **Export Single ATK Stock Request**: Allows users to export details of a single ATK Stock Request.
2.  **Bulk Export ATK Stock Requests**: Allows users to export details of multiple ATK Stock Requests.

These features will enhance reporting and data management capabilities for ATK Stock Requests within the application.
